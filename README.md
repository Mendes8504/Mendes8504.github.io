# 👨‍💻 Meu Portfólio Profissional

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades como desenvolvedor, meus projetos práticos e minha evolução na área de tecnologia.

## 🎯 Sobre o Projeto

Este portfólio web foi construído do zero com foco em uma arquitetura limpa, código organizado e padrões profissionais. A interface foi desenhada pensando em tendências modernas de UI (como efeitos de *glassmorphism*) e responsividade, garantindo uma ótima experiência tanto em dispositivos móveis quanto em desktops.

Atualmente, sou estudante do 3º semestre de Análise e Desenvolvimento de Sistemas (ADS) e busco oportunidades para aplicar meus conhecimentos em desenvolvimento de software e criação de soluções eficientes.

## 🛠️ Tecnologias Utilizadas

A base deste projeto foi construída com as seguintes tecnologias:

*   **HTML5:** Estruturação semântica e acessível.
*   **CSS3:** Estilização avançada, animações, efeitos visuais modernos e layout responsivo (Mobile-First).
*   **JavaScript:** Lógica de interface, interatividade e dinamismo da página.

*(Nota: Minha stack de desenvolvimento também inclui **Python** para lógica de back-end e automação em outros projetos práticos).*
